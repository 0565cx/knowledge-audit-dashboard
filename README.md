## 后续新增看板

建议每个完整审核看板使用独立目录，例如：

```text
full-dashboards/mechanical-engineer/
full-dashboards/electrical-engineer/
full-dashboards/automation-engineer/
```

新增后，在根目录 `index.html` 中添加对应入口卡片。
